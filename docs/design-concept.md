# Дизайн-концепция MyRec

- **Стиль:** clean/minimal + food magazine + premium UI.
- **Палитра:** cream background, espresso text, saffron CTA, terracotta accents.
- **Типографика и spacing:** большие заголовки, воздушные отступы, округлые карточки `rounded-3xl`.
- **Анимации:**
  - reveal on scroll (`MotionSection` + Framer Motion),
  - hover lift + image zoom на карточках,
  - плавные transition на кнопках.
- **Hero:** крупное food-image + затемнение + сильный копирайт.
- **Админка:** визуально спокойная, четкие таблицы и формы.
